<h2 align="center">✨ Commands ✨</h2>

- 👁️ **View Task**: Want to view details about your task? Just type `view` and after that, the task's index.
- ➕ **Add Task**: With the `add` command you can add your own task.
- ✏️ **Edit Task**: Switch between light and dark themes effortlessly.
- 📃 **Change Task's Status**: Just type `status`, the index of the task and the new status.
- ❌ **Delete Task**: Not happy with some task? Delete it with `delete` and its index.
- 🗑️ **Delete All Tasks**: Starting from scratch? Gotcha, `delete all`.
- 🧹 **Clear Terminal**: Anything bothering you? Clear the CLI with `clear`.
- 📱 **Exit**: Exiting the application? Easy, type `exit`.

<h2 align="center">🛠 Installation Guide 🛠</h2>

1. **Clone the repository**:

   ```bash
   git clone https://github.com/munraitoo13/godo-list
   ```

2. **Run the app**:

   ```bash
   go run ./cmd
   ```
